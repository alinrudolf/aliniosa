# Personal System Interface

A personal presentation website built with Vite, React, TypeScript, TailwindCSS, CSS variables, and SVG-based visuals.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Structure

Content lives in `src/data`. Visual and layout primitives live in `src/components`. Design tokens live in `src/styles/tokens.css` and are imported once from `src/main.tsx`.
